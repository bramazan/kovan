# Installation

## 1- Node.JS Server

Clone this repo and go to server folder.

```
$ cd server
$ npm install
```

### Run the app

```
$ npm run start:dev
```

and go to [localhost:4000/graphql](http://localhost:4000/graphql)

## 2- React Client

Go to client folder.

```
$ cd client
$ npm install
```

### Run the app

```
$ npm start
```

and go to [localhost:3000](http://localhost:3000/)
